## Capstone Project Options

Pilih salah satu project di bawah, atau propose project sendiri (dengan approval mentor):

### Option A: E-Commerce Store
- Product listing dengan categories dan search
- Shopping cart (add, remove, update quantity)
- User auth (register, login, profile)
- Checkout flow (order summary, order history)
- Admin panel: manage products (CRUD)

### Option B: Social Media / Forum
- User registration dan profiles
- Create posts dengan text dan images
- Like dan comment system
- Follow/unfollow users
- News feed (posts dari followed users)

### Option C: Project Management Tool
- Create projects dengan multiple boards
- Kanban board dengan drag-and-drop cards
- Assign tasks ke team members
- Due dates dan status tracking
- Team invitation system

### Option D: Content Management System (CMS)
- Rich text editor untuk creating articles
- Categories dan tags
- Public blog view
- Admin dashboard dengan analytics
- User roles (admin, editor, viewer)

### Option E: Custom Project (Propose)
- Harus include: auth, CRUD, minimum 3 database tables, deployment
- Submit proposal ke mentor untuk approval dalam 2 hari pertama

## Demo Day Format

Setiap tim/individu punya **10 menit** untuk presentasi:

| Bagian | Durasi | Detail |
|--------|--------|--------|

| Solution Overview | 1 menit | Gimana solusinya? |
| Live Demo | 5 menit | Tunjukin app-nya jalan! |
| Technical Architecture | 2 menit | Tech stack, database schema, API design |
