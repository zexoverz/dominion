### Key Lessons
- Railway `bind` valid values: auto, lan, loopback, custom, tailnet (NOT "all")
- Railway rootDirectory must match the repo path exactly (e.g., `dominion/api` not just `api`)
- Railway project-scoped tokens can't do `railway up` CLI, but GraphQL mutations work
- `serviceConnect` mutation links a Railway service to a GitHub repo for auto-deploy
- Yuyu-tei Pokemon card images: `/poc/100_140/{set}/{id}.jpg` (different from OP `/opc/front/`)
- Yuyu-tei search results page doesn't return card images in the HTML directly (need to visit individual card pages)
- The "parallel" versions on Yuyu-tei have different IDs (e.g., OP13 Gorosei regular vs 特別パラレル)
